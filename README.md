**TROLL BOT 69** is a desi, witty, high-level trolling AI built to roast, crack dark jokes, and deliver double-meaning humor. Powered by Google's Gemini API, this bot understands Urdu, Roman Urdu, English, and Hinglish.

## 🚀 Features

### 🎭 9 Dynamic Modes
1. **LIGHT:** Mild roasts and basic slang.
2. **MEDIUM:** Funny insults and cultural references.
3. **HIGH:** Personal and brutal roasts.
4. **DARK:** Existential and dark humor.
5. **PRO:** Extreme language (Educational context).
6. **DICTIONARY:** Meanings for specific *gaali* (slang) words.
7. **EMOTIONAL:** Soft trolling with an emotional touch.
8. **RANDOM:** Surprise mode!
9. **DOUBLE DARK (GAND MODE):** 18+ double-meaning jokes with a dark twist.

### ⌨️ Quick Commands
- `/roast` - Get roasted instantly.
- `/joke` - Hear a joke without any fluff.
- `/dark` - Instantly switch to DARK mode.
- `/gand` or `/dmj` - Instantly switch to DOUBLE DARK (18+) mode.
- `/gaali [word]` - Get the "educational" meaning of a slang word.
- `/enough` - Silence the bot for 15 minutes.
- `/help` - View all commands.

## 💻 Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/troll-bot-69.git
   cd troll-bot-69
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:3000` (or the port provided in your terminal).
